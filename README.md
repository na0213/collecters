# collecters

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - みんなの好きなものをカテゴリーごとに登録できるアプリ

## 工夫した点

  - ログイン機能をつけた
  - ログインしてから、それぞれ好きなカテゴリー（本や切手など）で登録できる
  - ユーザー登録用テーブルと、カテゴリーテーブルを作成し、user_idが一致できるようにした
  - 各々が登録した内容は、共有として他の人も見ることができる

## 苦戦した点

  - ログインした後のマイページにて、user_idが一致したものを取得するところ
